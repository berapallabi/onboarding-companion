# Technology Stack

This project uses modern, high-performance tools to deliver a premium user experience.

## Frontend
- **Next.js 15**: The foundation, utilizing App Router, Server Components, and Server Actions.
- **React 19**: Modern UI library.
- **Tailwind CSS**: Utility-first styling with a custom "Neo-Arcade" theme.
- **Framer Motion**: Smooth micro-animations and page transitions.
- **Lucide React**: For consistent iconography.

## Backend & Persistence
- **PostgreSQL**: Robust relational database for all user and curriculum data.
- **Drizzle ORM**: Type-safe database interactions with automatic migration generation.
- **NextAuth.js (v5)**: Simplified authentication with GitHub/Google providers.

## AI & Intelligence
- **Vercel AI SDK**: Unified interface for LLM interactions.
- **OpenAI / Gemini**: Underlying LLM providers for the chat companion.
- **RAG (Retrieval-Augmented Generation)**: Custom implementation for project-specific knowledge.

## Development Tools
- **TypeScript**: Full type safety across the entire stack.
- **Drizzle Kit**: CLI tools for database prototyping and migrations.
- **TSX**: For executing TypeScript scripts (like the database seeder).
- **ESLint/Prettier**: Code quality and formatting.
