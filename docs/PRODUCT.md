# Product Overview

## Vision
The **Smart Onboarding Companion** is designed to transform the often-overwhelming first weeks of a new job into a gamified, personalized, and efficient journey. By leveraging AI and role-specific data, we provide a tailored experience that reduces "time-to-productivity" and increases new hire engagement.

## Core Features

### 1. Personalized Assessment
- **Role Selection**: Tailors the entire experience to Engineering, Design, Product, or Marketing.
- **Tech Stack Mapping**: Users select their specific tools (e.g., React, Kubernetes, Figma, Jira).
- **Goal Alignment**: Captures user aspirations to personalize the quest log.

### 2. The Dashboard (The Navigator)
- **Focused Milestones**: A weekly sequence of tasks categorized by Role Foundations and Tech Deep Dives.
- **Progress Tracking**: Real-time visual feedback on onboarding completion.
- **Dynamic Focus Area**: Highlights the specific technologies the user is currently mastering.

### 3. AI Onboarding Companion
- **Context-Aware Assistance**: An RAG-powered chat interface that answers questions based on internal documentation.
- **Natural Language Quests**: Users can ask the AI to generate custom learning paths or explain complex internal jargon.

### 4. Training Room & Quests
- **Knowledge Checks**: Role-specific quizzes to validate learning and "level up."
- **Quest Log**: Gamified tasks that reward users with XP for completing real-world onboarding activities.
- **Role-Aware Challenges**: Specialized challenges tailored to the user's selected tech stack.

### 5. Knowledge Base
- **Centralized Docs**: Accessible portal for all internal documentation.
- **Smart Search**: Filters resources based on current milestones and user queries.
